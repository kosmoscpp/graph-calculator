# 📈 Python Graphing Calculator

A simple Python-based graphing calculator that allows users to enter any mathematical expression in terms of `x` and visualize its graph.

---

## ✨ Features

- Supports mathematical expressions such as:
  - `sin(x)`
  - `cos(x)`
  - `exp(x)`
  - `log(x)`
  - `x**2 + 3*x - 5`
  - more...
- Interactive graph plotting using **Matplotlib**
- Customizable x-axis range
- Easy to use and beginner-friendly

## ⚙️ Installation & Usage

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/graphing-calculator.git
cd graphing-calculator
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Program

```bash
python graph_calculator.py
```

---

## 📊 Example

### Input

```text
Enter expression in x: sin(x) + x**2
Enter x range (min max): -10 10
```

### Output

👉 A graph window will open displaying the plotted function.
